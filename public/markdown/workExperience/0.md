    - 정보 수정 및 삭제 시, 변경사항이 있는 객체만 서버에 전달하여 사용성 개선
    - 라이브러리 내 유효성 검사 기능을 통한 에러 처리로 사용자 경험 개선
  - `ApexChart`를 통한 데이터 집계 그래프 구현

  - 유저 권한에 따른 `routing` 적용
